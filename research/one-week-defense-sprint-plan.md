# 答辩前一周“冲刺”执行计划（One-Week Sprint Plan）

鉴于距离答辩仅剩一周，且我们已确立了**“展示三大基础能力，以此引出两项结合计划”**的务实汇报策略，接下来必须将长期规划（`piml-matrix-free-execution-plan.md`）中的初期目标极度压缩，形成一个聚焦于**“产出可运行原型与可视化结果”**的一周可执行方案。

该计划的核心原则是：**不追求工业级全流程的完整度，只追求核心逻辑的跑通与高质量的学术图表输出**。

## 一周冲刺计划日程表 (Day 1 - Day 7)

### Day 1-2：展示能力 A —— Matrix-Free 高性能扩展性曲线
*依托博士期间积累，快速产生性能对比数据*
- **任务目标**：利用已有的 FEALPy / SOPTX 基础，针对二维/三维 MBB 梁问题，分别运行基于直接求解器（全组装）和基于 Matrix-Free CG 的线性方程组求解。
- **具体执行**：
  1. 编写测试脚本 `benchmark_matrix_free.py`。
  2. 构造规模序列（自由度从 $10^4$ 逐渐增大到 $10^7$）。
  3. 记录时间与内存开销，并在直接求解器发生 OOM (Out of Memory) 时捕获节点。
  4. 使用 `matplotlib` 绘制极具说服力的扩展性对比图（保存为 `mf_scaling.pdf`）。

### Day 3-4：展示能力 B —— PIML 多尺度预测流程原型
*抽取长期计划 T1.3.1 的核心，搭建不带优化的“前向预测验证”*
- **任务目标**：跑通“宏观密度 -> (模拟 PIML) -> 微观刚度矩阵 -> 组装宏观方程”的单步前向流程。
- **具体执行**：
  1. 编写脚本 `piml_baseline_forward.py`。
  2. 离线端（Mock）：无需真花几天时间训练复杂网络，可使用一个极简的多层感知机（MLP）或者直接写死几个查表映射，模拟 PIML 的预测行为。
  3. 在线端：在 FEALPy 的二维粗网格上，输入宏观密度，调用上述模型获得等效刚度矩阵。
  4. 绘制宏观粗网格与局部细观单元的映射关系图（保存为 `piml_baseline.pdf`），向评委证明“PIML 的核心通信管道已经打通”。

### Day 5-6：展示能力 C —— MMC 显式几何与高阶网格切割
*截取长期计划 T3.1.2 的开头，仅做几何与数值积分的展示，不跑灵敏度*
- **任务目标**：在 FEALPy 中实现一个背景网格，并将一个显式组件（如带角度的胶囊体）覆盖其上，识别被切割的网格并展示高阶高斯积分点。
- **具体执行**：
  1. 编写脚本 `mmc_cut_mesh.py`。
  2. 定义一个高阶拉格朗日背景网格，以及一个 MMC 组件方程。
  3. 实现基本的内外判定逻辑，提取出被边界切割的单元。
  4. 画出被切割单元内的积分点分布规律，区分材料点与孔洞点（保存为 `mmc_baseline.pdf`），向评委证明“显式几何如何与高阶数值离散结合的难题已具备解法思路”。

### Day 7：PPT 整合、排版与最终演练
- **任务目标**：将前六天生成的真实学术图表整合进 `template-8min.tex`，并固化演讲逐字稿。
- **具体执行**：
  1. 更新 `outline-8min.md` 大纲，将三大能力的讲述串联起来。
  2. 在 LaTeX 中修改排版，放置上述生成的三张图表，编译出最终版 PDF。
  3. 全程模拟演练，确保时间控制在 8 分钟内。
