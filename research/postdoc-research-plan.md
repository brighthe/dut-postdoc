# 博士后科研计划

## 研究计划题目

1. 面向大规模结构拓扑优化的 PIML 与 Matrix-Free 高性能求解方法研究
2. 面向大规模结构拓扑优化的 GPU 并行有限元分析与快速求解技术研究
3. 面向 MMC/MMV 结构拓扑优化的高精度数值离散方法与快速算法研究

## 博士后科研计划名称

面向大规模结构拓扑优化的 PIML 与 Matrix-Free 高性能求解方法研究

---

## 一、工作计划

本研究拟围绕大规模结构拓扑优化中的有限元快速分析、高性能迭代求解与复杂结构优化建模问题，开展 PIML、Matrix-Free、GPU 并行计算以及先进数值离散方法与 MMC/MMV 显式拓扑优化方法相结合的研究工作。

### 第一阶段：调研与原型复现

系统调研 PIML、Matrix-Free 有限元快速求解、GPU 并行计算、MMC/MMV 拓扑优化方法以及胡张混合元、虚单元等先进数值离散方法的发展现状，明确大规模结构拓扑优化中有限元分析、灵敏度计算和优化迭代的主要计算瓶颈。结合团队现有 FEALPy、SOPTX 等软件基础，复现典型 PIML 与 Matrix-Free 算法流程，建立可验证的基础算例和原型程序。

### 第二阶段：PIML 与 Matrix-Free 协同求解

重点研究面向结构拓扑优化的 PIML 与 Matrix-Free 协同求解方法。通过 PIML 学习或构造具有问题无关特征的局部力学算子、多尺度形函数或等效刚度表示，并将其嵌入 Matrix-Free 有限元分析与迭代求解框架中，减少全局矩阵组装和存储开销，提高大规模拓扑优化中的结构分析效率。在此基础上，进一步开展 GPU 并行实现与加速研究，形成适用于大规模结构拓扑优化的高性能计算原型。

### 第三阶段：先进数值方法与 MMC/MMV 结合

结合本人在高阶有限元、胡张混合元和拓扑优化算法实现方面的研究基础，探索胡张混合元、虚单元等先进数值方法在 MMC/MMV 显式拓扑优化框架中的应用，重点研究复杂几何结构、高精度应力分析、几何非线性结构优化等问题中的数值离散、灵敏度分析和快速求解方法。最终形成较完整的算法体系、软件模块和典型算例验证，为团队结构拓扑优化软件平台建设和后续工程应用提供支撑。

## 二、研究思路

本研究的基本思路是：以大规模结构拓扑优化中的结构分析效率和复杂结构优化能力为核心问题，围绕"**问题无关机器学习建模 → Matrix-Free 快速求解 → GPU 并行加速 → 先进数值方法与显式拓扑优化结合**"开展系统研究。

**第一条主线：PIML 问题无关力学算子构造**

针对传统拓扑优化中每一步迭代都需要反复进行有限元分析、全局刚度矩阵组装和线性方程组求解所带来的高计算成本问题，研究基于 PIML 的问题无关力学算子构造方法。该方法不直接针对某一具体载荷、边界条件或优化构型训练端到端模型，而是面向局部子结构、粗网格单元或典型材料分布形式，学习可迁移的局部力学表示，从而服务于不同优化问题中的快速结构分析。

**第二条主线：Matrix-Free + GPU 并行加速**

将 PIML 构造的局部力学算子与 Matrix-Free 有限元快速求解框架相结合。在求解过程中避免显式组装和存储大规模全局刚度矩阵，而是通过局部单元计算、算子作用和迭代求解实现矩阵-向量乘积计算，从而降低内存消耗并提高求解效率。进一步结合 GPU 并行计算技术，对单元积分、算子作用、灵敏度分析和优化迭代过程进行并行化设计，提升大规模三维结构拓扑优化问题的计算能力。

**第三条主线：先进数值方法与 MMC/MMV 结合**

面向复杂结构优化和团队 MMC/MMV 显式拓扑优化方向，研究胡张混合元、虚单元、高阶有限元等先进数值离散方法与拓扑描述方法的结合机制。一方面利用先进数值方法提高复杂几何、应力分析和几何非线性问题中的结构分析精度；另一方面结合 MMC/MMV 方法边界清晰、变量维数较低和几何描述明确的特点，建立适用于复杂结构拓扑优化的高精度、高效率数值算法框架。

---

## 三、预期目标

**目标一：PIML + Matrix-Free 协同求解方法**

建立面向大规模结构拓扑优化的 PIML 与 Matrix-Free 协同求解方法，形成可用于有限元快速分析和拓扑优化迭代的算法原型，并通过典型二维、三维结构算例验证其有效性和计算效率。

**目标二：GPU 并行有限元分析与快速求解模块**

形成面向结构拓扑优化的 GPU 并行有限元分析与快速求解模块，重点实现 Matrix-Free 算子作用、迭代求解、灵敏度分析和优化更新过程的并行加速，提高大规模结构优化问题的求解规模和计算效率。

**目标三：先进数值方法与 MMC/MMV 结合框架**

探索胡张混合元、虚单元等先进数值方法与 MMC/MMV 显式拓扑优化方法的结合，建立适用于复杂几何、高精度应力分析和几何非线性结构优化问题的数值计算框架，形成若干具有代表性的数值算例。

**目标四：软件模块与学术产出**

依托团队现有 FEALPy、SOPTX 及相关软件平台，完成若干可复用的算法模块和算例程序，支撑团队结构拓扑优化软件平台建设。研究期间力争发表高水平学术论文，参与相关科研项目申报与软件平台开发。
