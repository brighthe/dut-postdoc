# 博士后科研计划

## 研究计划题目

1. 面向大规模结构拓扑优化的 PIML 与 Matrix-Free 高性能求解方法研究
2. 面向大规模结构拓扑优化的 GPU 并行有限元分析与快速求解技术研究
3. 面向 MMC/MMV 结构拓扑优化的高精度数值离散方法与快速算法研究

## 博士后科研计划名称

面向大规模结构拓扑优化的 PIML 与 Matrix-Free 高性能求解方法研究

---

## 一、工作计划

本研究拟围绕大规模结构拓扑优化中的有限元快速分析、高性能迭代求解与复杂结构优化建模问题，面向合作导师团队在结构拓扑优化、MMC/MMV 显式拓扑描述、PIML 方法、国产自主 CAE 软件平台及工程应用等方面的研究基础，开展 PIML、Matrix-Free、GPU 并行计算以及先进数值离散方法与显式拓扑优化方法相结合的研究工作。研究过程中，将结合 SiPESC 工程与科学计算集成化软件平台、SiPESC.FEMS 结构有限元分析系统、SiPESC.TOPO 结构拓扑优化子系统以及 DLUTopt 拓扑优化软件平台的已有基础，充分发挥本人在计算数学、数值 PDE、有限元方法、拓扑优化算法和科学计算软件实现方面的研究积累，逐步形成面向大规模结构拓扑优化的高效分析求解方法、软件算法原型和典型算例验证。

### 第一阶段：文献调研、平台熟悉与算法原型验证

系统调研 PIML、Matrix-Free 有限元快速求解、GPU 并行计算、MMC/MMV 拓扑优化方法以及胡张混合元、虚单元等先进数值离散方法的发展现状，明确大规模结构拓扑优化中结构分析、灵敏度计算和优化迭代的主要计算瓶颈。

在此基础上，重点熟悉 SiPESC 工程与科学计算集成化软件平台、SiPESC.FEMS 结构有限元分析系统、SiPESC.TOPO 结构拓扑优化子系统以及 DLUTopt 拓扑优化软件平台，梳理合作导师团队在 MMC/MMV 显式拓扑优化、有限元求解器、结构优化流程集成、前后处理及工程算例验证等方面的已有基础。结合本人在计算数学、有限元数值方法、拓扑优化算法实现和科学计算软件开发方面的前期积累，复现并验证典型 PIML 与 Matrix-Free 算法流程，建立可用于后续研究的基础算例、算法原型和性能评估框架。

### 第二阶段：PIML 与 Matrix-Free 协同求解方法研究

重点研究面向结构拓扑优化的 PIML 与 Matrix-Free 协同求解方法。依托本人在数值 PDE、有限元离散、误差分析和迭代求解算法方面的计算数学基础，分析 PIML 局部力学算子、多尺度形函数或等效刚度表示的数值稳定性、可迁移性和适用范围，使其能够面向不同载荷、边界条件和优化构型提供可复用的局部力学近似模型，从而降低拓扑优化迭代过程中反复结构分析的计算成本。

进一步将 PIML 构造的局部力学算子嵌入 Matrix-Free 有限元分析与迭代求解框架中，避免显式组装和存储大规模全局刚度矩阵，通过单元级算子作用、矩阵-向量乘积和迭代求解实现高效结构分析。在此基础上，面向大规模三维结构拓扑优化问题，开展 GPU 并行实现与加速研究，对单元积分、算子作用、迭代求解、灵敏度分析和优化更新等关键环节进行并行化设计，形成适用于大规模结构拓扑优化的高性能计算原型。

### 第三阶段：先进数值方法与 MMC/MMV 显式拓扑优化结合

结合本人在高阶有限元、胡张混合元、变密度拓扑优化和科学计算软件实现方面形成的研究基础，面向合作导师团队 MMC/MMV 显式拓扑优化方向，探索胡张混合元、虚单元等先进数值离散方法与显式拓扑描述方法的结合机制。重点研究复杂几何结构、高精度应力分析、几何非线性结构优化等问题中的数值离散、灵敏度分析和快速求解方法。

在此基础上，进一步结合 MMC/MMV 方法边界描述清晰、设计变量维数较低、便于与 CAD/CAE 流程衔接等特点，研究适用于复杂结构优化问题的高精度结构分析与优化建模方法。围绕典型航空航天、装备结构和复杂承力构件算例，开展算法验证、性能测试和工程适用性分析，逐步形成较完整的算法体系、软件模块和典型算例库，为合作导师团队结构拓扑优化软件平台建设和后续工程应用提供支撑。

## 二、研究思路

本研究的基本思路是：以大规模结构拓扑优化中的结构分析效率、迭代求解性能和复杂结构优化建模能力为核心问题，围绕“**问题无关机器学习建模 → Matrix-Free 快速求解 → GPU 并行加速 → 先进数值方法与显式拓扑优化结合 → 软件平台验证与工程算例支撑**”开展系统研究。研究过程中，将面向合作导师团队在 SiPESC 工程与科学计算集成化软件平台、SiPESC.FEMS 结构有限元分析系统、SiPESC.TOPO 结构拓扑优化子系统以及 DLUTopt 拓扑优化软件平台等方面的已有基础，结合本人在计算数学、数值 PDE、有限元方法、拓扑优化算法和科学计算软件实现方面的研究积累，形成兼具理论分析、算法设计、程序实现和算例验证的研究路线。

**第一条主线：PIML 问题无关力学算子构造**

针对传统拓扑优化中每一步迭代都需要反复进行结构分析、全局刚度矩阵组装和线性方程组求解所带来的高计算成本问题，研究基于 PIML 的问题无关力学算子构造方法。该方法不直接针对某一具体载荷、边界条件或优化构型训练端到端模型，而是面向局部子结构、粗网格单元或典型材料分布形式，学习或构造可迁移的局部力学表示，包括多尺度形函数、等效刚度表示或局部算子近似等，从而服务于不同优化问题中的快速结构分析。

在此过程中，充分发挥本人在计算数学方向的研究基础，重点关注 PIML 局部力学算子的数值稳定性、适用范围、误差传播特征以及与有限元离散格式之间的匹配关系，避免仅停留在数据驱动替代计算的层面，而是从数值方法和力学算子构造角度提升其可靠性和可解释性。

**第二条主线：Matrix-Free 快速求解与 GPU 并行加速**

将 PIML 构造的局部力学算子与 Matrix-Free 有限元快速求解框架相结合。在求解过程中避免显式组装和存储大规模全局刚度矩阵，而是通过单元级计算、算子作用、矩阵-向量乘积和迭代求解实现高效结构分析，从而降低内存消耗并提高大规模问题的求解效率。

进一步面向大规模三维结构拓扑优化问题，研究 GPU 并行计算环境下的 Matrix-Free 算子作用、迭代求解、灵敏度分析和优化更新方法。重点对单元积分、局部算子计算、并行矩阵-向量乘积、预条件迭代求解及优化迭代流程进行协同设计，形成适用于大规模结构拓扑优化的高性能计算方法。

**第三条主线：先进数值方法与 MMC/MMV 显式拓扑优化结合**

面向复杂结构优化和合作导师团队 MMC/MMV 显式拓扑优化方向，研究胡张混合元、虚单元、高阶有限元等先进数值离散方法与显式拓扑描述方法的结合机制。一方面，利用先进数值方法提高复杂几何、高精度应力分析和几何非线性问题中的结构分析精度；另一方面，结合 MMC/MMV 方法边界描述清晰、设计变量维数较低、便于与 CAD/CAE 流程衔接等特点，建立适用于复杂结构拓扑优化的高精度、高效率数值算法框架。

该主线将重点体现本人在有限元数值方法、混合有限元、变密度拓扑优化和算法实现方面的前期积累，探索计算数学方法与显式拓扑优化方法之间的结合点，为复杂承力结构、高精度应力约束优化和工程化拓扑优化软件开发提供支撑。

**第四条主线：软件平台适配、算例验证与工程应用支撑**

在算法研究的基础上，进一步面向 SiPESC、SiPESC.FEMS、SiPESC.TOPO 以及 DLUTopt 等软件平台的已有功能体系和算例需求，开展算法原型实现、模块化封装和典型算例验证工作。重点围绕有限元分析、拓扑优化流程、前后处理接口、工程算例验证和性能评估等环节，形成可复用的算法模块和算例程序，为合作导师团队结构拓扑优化软件平台建设、工程应用拓展和后续科研项目申报提供支撑。

## 三、预期目标

**目标一：建立 PIML 与 Matrix-Free 协同求解方法**

建立面向大规模结构拓扑优化的 PIML 与 Matrix-Free 协同求解方法，形成可用于有限元快速分析和拓扑优化迭代的算法原型。重点实现问题无关局部力学算子、多尺度形函数或等效刚度表示与 Matrix-Free 有限元迭代求解框架的结合，并通过典型二维、三维结构算例验证其有效性、稳定性和计算效率。

**目标二：形成 GPU 并行有限元分析与快速求解原型**

形成面向结构拓扑优化的 GPU 并行有限元分析与快速求解原型，重点实现 Matrix-Free 算子作用、矩阵-向量乘积、预条件迭代求解、灵敏度分析和优化更新过程的并行加速。通过对比传统组装式有限元分析方法，评估所提方法在内存消耗、求解规模和计算效率方面的改进效果，提高大规模结构优化问题的求解能力。

**目标三：构建先进数值方法与 MMC/MMV 显式拓扑优化结合框架**

探索胡张混合元、虚单元、高阶有限元等先进数值方法与 MMC/MMV 显式拓扑优化方法的结合，建立适用于复杂几何结构、高精度应力分析和几何非线性结构优化问题的数值计算框架。形成若干具有代表性的数值算例，验证先进数值离散方法在提升结构分析精度、改善应力计算质量和支撑复杂结构优化方面的作用。

**目标四：形成可复用算法模块和典型算例体系**

面向 SiPESC、SiPESC.FEMS、SiPESC.TOPO 以及 DLUTopt 等软件平台的已有基础和发展需求，完成若干可复用的算法模块、原型程序和典型算例，支撑结构拓扑优化软件平台建设。相关模块将重点服务于有限元快速分析、Matrix-Free 迭代求解、GPU 并行加速、显式拓扑优化和工程算例验证等环节。

**目标五：支撑学术论文、项目申报与工程应用拓展**

围绕 PIML 与 Matrix-Free 协同求解、GPU 并行快速分析、先进数值方法与 MMC/MMV 显式拓扑优化结合等研究内容，力争形成具有创新性的学术论文成果，并参与相关科研项目申报和软件平台开发工作。通过典型航空航天、装备结构和复杂承力构件算例验证研究成果的适用性，为合作导师团队在结构拓扑优化理论方法、国产 CAE 软件平台建设和工程应用推广方面提供支撑。
