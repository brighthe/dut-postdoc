# 2026 入站考核 · 科研工作汇报（Beamer 幻灯片）

## 答辩安排

- **场景**：大连理工大学力学与航空航天学院博士后进站集中考核。
- **时间**：2026 年 7 月第一周（石圣哲老师 2026-06-23 在「7月博后集中考核群」通知）。
- **汇报要求**：个人汇报 PPT，时长约 8 分钟。
- **材料状态**：进站材料基本齐；后续需确认/补交「业绩一览表」。材料准备好后微信发石老师，由其打印纸质版。
- **注意**：业绩一览表口径存在小矛盾——此前电子版已发且石老师曾说由大工打印/填写，6 月 23 日群通知又点名“何老师需业绩一览表”。制作答辩材料前应先确认是沿用旧版还是按网页模板补一版。

## 如何编译

用 **XeLaTeX**（支持中文）编译，编译**两次**以生成目录：

- **TeXstudio**：打开 `template-8min.tex`，默认编译器选 XeLaTeX，点编译即可（按两次或用"构建并查看"）。
- **命令行**：`xelatex template-8min.tex` 执行两遍。

## 注意

编译产生的中间文件（`*.aux` / `*.log` / `*.nav` / `*.snm` / `*.toc` /
`*.synctex.gz` 等）和成品 `template-8min.pdf` 都已被仓库根目录 `.gitignore` 忽略，
直接在本目录编译即可，不会污染版本控制。`figures/` 下的矢量图正常跟踪。
