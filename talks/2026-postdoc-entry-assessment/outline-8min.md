## 帧 4 · 方法三 · 任意次胡张混合有限元

**核心一句话**：以应力为独立未知量、构造 $H(\mathrm{div})$ 协调对称应力空间的任意次胡张混合元，嵌入变密度拓扑优化闭环，直接获得高精度应力场。

**配图**：抗闭锁对比 `bearing_disp_nu4999` vs `bearing_mixed_nu4999`（二维轴承，近不可压 $\nu_0=0.4999$）；如版面够，附应力场 `stress_vm_disp` vs `stress_vm_mixed` 小图。

**左：方法（精简）**
- 以应力 $\boldsymbol\sigma$ 为独立未知量，构造 $H(\mathrm{div})$ 协调对称张量有限元空间（北大胡俊教授方法，完成程序实现并引入拓扑优化）；
- 基于单纯形晶格统一实现任意阶次；每步迭代同时求解 $\boldsymbol\sigma_h,\boldsymbol u_h$，直接由 $\boldsymbol\sigma_h$ 构造柔顺度目标与局部应力约束。

**右：两项代表性能力（数据）**
- **抗体积闭锁**：近不可压 $\nu_0=0.4999$ 下，标准位移法刚度高估、构型异常（$c\approx55.78$）；胡张混合法构型稳定（$c\approx102.79$）。
- **应力场可靠评估**：位移法应力场单元间非物理跳跃；混合法基于协调应力空间，应力场平滑连续、约束评估更可靠（悬臂梁 V 型承载构型）。

**底部结论条**：实现"胡张混合分析 → 变密度拓扑优化闭环"系统拓展，为高精度应力驱动拓扑优化奠定基础。

**讲稿要点**：抗闭锁 + 应力评估两个亮点各一句。原"复杂边界与低阶稳定化"帧**删去**（技术细节，口头一句带过即可）。

- **架构三层**：用户交互层 / 核心算法层（插值-分析-正则化-优化闭环）/ FEALPy 基础后端层；
- **关键技术**：高效张量化矩阵组装、PyTorch/JAX 自动微分灵敏度、NumPy/PyTorch/JAX 多后端与 CPU/GPU 异构；
- **性能亮点（GPU 加速表，三维悬臂梁）**：总耗时 2789.9 s → **636.7 s**，端到端 **≈4.38×**；稳态平均迭代 21.09 s → 4.84 s，**≈4.36×**；
- 多后端构型/收敛一致（NumPy/PyTorch/JAX）；AD 与手动微分结果一致、效率相当、扩展更便捷（口头一句）。

**底部结论条**：模块化解耦 + AD + 多后端异构，为高阶有限元拓扑优化提供高效可扩展软件支撑。

**讲稿要点**：讲"解耦 + GPU 4.38× + 多后端一致"三点即可，~40 秒。

## 帧 7 · 方向一 · PIML 增强多尺度分析 + Matrix-Free 高性能求解

**核心一句话**：面向大规模拓扑优化中反复细网格分析的高成本，用 PIML 在线预测多尺度形函数、构造粗尺度等效刚度，并进一步发展全局算子 Matrix-Free 高性能求解。

**配图**：【前期基础】PIML 架构与 Matrix-Free 求解原型（已补充 PIML 宏微观映射示意图与 Matrix-Free 性能对比曲线）。

**主线 1：PIML 多尺度形函数预测与粗尺度等效刚度**
- 以粗网格单元/子结构内部已知细尺度材料密度为输入，离线训练 PIML 模型在线预测多尺度形函数，减少重复求解局部细尺度问题；
- 重点研究等效刚度的对称性、正定性、能量一致性、误差传播；分析粗细尺度比、局部边界条件、预测误差的影响。

**主线 2：从无持久存储到全局算子 Matrix-Free**
- 复现并优化"按需预测—算后释放 + 全局缩聚刚度组装 + PETSc 迭代求解"基线流程；
- 进一步研究**不显式组装全局缩聚矩阵**的 Matrix-Free 算子作用，结合 Krylov 迭代 + 多层预条件，面向多核 CPU / GPU / 异构的并行高性能求解。

**讲稿要点**：一句点"PIML 替代重复局部求解"，一句点"Matrix-Free 把存储/规模瓶颈再降一层"，并指出右侧图示为基于 FEALPy 搭建的初步算法原型的运行结果，展示了 Matrix-Free $O(N)$ 的理想扩展性。

## 帧 8 · 方向二 · MMC 显式拓扑优化先进数值分析

**核心一句话**：面向 MMC/MMV 显式几何描述与分析模型的耦合，研究高精度数值离散与高效结构分析方法，结合高阶拉格朗日元或胡张混合元提升求解精度。

**配图**：【前期基础】MMC 边界切割与高阶离散积分原型（已补充显式组件切割背景网格并进行高阶 Gauss 积分的示意图）。

**主线 3：MMC 高精度数值离散**
- 研究几何边界投影、边界切割单元处理、数值积分与灵敏度的一致性，评估几何近似/离散误差影响；
- 将**高阶拉格朗日有限元、胡张混合有限元**与 MMC 显式描述直接结合，有效处理边界切割单元，提升复杂几何与应力约束问题的精度与稳定性（直接承接博士方法一、三）。

**主线 4：MMC 高效结构分析**
- 利用 MMC/MMV 边界清晰、设计变量维数低、几何演化显式可控的特点，研究分析模型高效更新与快速求解；
- 集成多分辨率离散、自适应局部加密、子结构分析与求解信息复用；**将方向一的 Matrix-Free 迁移应用于 MMC/MMV**，实现两方面在求解技术上的相互支撑。

**讲稿要点**：强调"显式几何 + 先进高阶离散"，指出右侧初步结果展示了显式边界对网格切割时的精确积分处理策略，并点出第四主线借方向二的 Matrix-Free 形成闭环。

## 帧 9 · 工作计划 · 预期目标 · 平台依托

**核心一句话**：分三阶段推进，形成两套方法 + 一套可复用算法/软件模块体系，依托郭旭团队与算海团队的互补优势落地。

**三阶段工作计划（每段一句）**
1. **第一阶段**：文献调研、基准复现（全尺度/多尺度/PIML 增强分析、MMC/MMV 代表方法）、建立统一算例与性能评估框架；
2. **第二阶段**：PIML 多尺度分析与 Matrix-Free 求解方法研究 + MMC/MMV 先进数值分析方法研究；
3. **第三阶段**：并行实现与软件集成（CPU/MPI → GPU/异构）、典型二维/三维算例验证与比较。

**预期目标（三条，浓缩）**
- 目标一：建立 PIML 增强多尺度分析 + Matrix-Free 高性能求解方法（含稳定性/误差准则、并行原型）；
- 目标二：建立 MMC/MMV 高精度离散 + 高效结构分析方法（含一致灵敏度、虚单元/混合/高阶离散方案）；
- 目标三：形成算法原型、可复用软件模块与统一典型算例体系，衔接 FEALPy / SiPESC / DLUTopt。

**平台依托（一行）**：郭旭院士团队（力学建模、MMC/MMV、PIML、SiPESC / DLUTopt 工业软件）× 算海团队（计算数学、FEALPy）——力学方法与数值计算互补。

**讲稿要点**：阶段讲节奏、目标讲产出、平台讲底气，收束到"理论—算法—程序—验证"完整链条。
